Chart.js 系列课程

第 7 课：散点图与气泡图

学习如何用 x/y 坐标表示二维关系，以及如何用 r 表示第三维大小。

本课你会学到什么

  • 散点图适合看两个变量的关系，例如学习时长和成绩、价格和销量。
  • 气泡图比散点图多一个 r 半径字段，用来表示第三个维度。
  • 本课要重点记住：scatter / bubble 的数据不是普通数组，而是对象数组。

图表示例

散点图

每个点都有一个 x 和 y 坐标。

气泡图

每个点除了 x 和 y，还多了一个 r 表示大小。

如何自定义成你自己的图表

  • 散点图要改自己的数据时，改成 { x: 数值, y: 数值 } 这种对象数组。
  • 气泡图则要写成 { x: 数值, y: 数值, r: 半径 }。
  • 如果你看不懂横轴和纵轴，请先在 scales.x.title 与 scales.y.title 里写清楚轴标题。
  • 当你想表达第三维（例如用户规模、项目体量）时，优先考虑气泡图。

课后练习建议

  1. 把散点图改成“学习时长与考试分数”的关系。
  2. 把气泡图改成“产品价格、销量、利润”的三维关系。
  3. 手动增加 3 个点，观察整体分布是否更集中。

# 故事协议文档索引

当前项目的故事数据协议说明已经拆分为两篇独立文档：

- [`故事协议V1.md`](故事协议V1.md)
- [`故事协议V2.md`](故事协议V2.md)

建议按需阅读：

- 如果你在维护旧故事、旧脚本文本或 `nextNode` 结构，请看 **V1 文档**。
- 如果你在编写新故事、使用编辑器导出、或让大模型生成结构化剧情，请看 **V2 文档**。

当前项目的文件组织推荐约定：

- 故事 JSON 放在 `stories/`
- 本地图片资源放在 `images/`
- 故事中的本地图片路径统一写成 `images/...`

说明：

- 这两篇文档被视为 **两套互不关联的独立协议说明**。
- 当前运行时同时支持读取两者，这是兼容实现，不代表两者属于同一套协议的不同语法层。

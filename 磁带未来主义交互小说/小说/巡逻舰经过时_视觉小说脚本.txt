《巡逻舰经过时》视觉小说脚本版

主要角色：
- 你
- 狼
- 设计师
- 方舟系统
- 广播

【序章：白色天空】

【场景 0-1】城市上空 / 白昼 / 阴冷
【画面】病白色天空，母舰悬在云层后，像一轮不会落下的月亮。
【音效】低沉嗡鸣，远处防空警报残响。

旁白：
地球被外星人入侵以后，最先改变的不是城市，而是天空。

旁白：
后来改变的是地球本身。

【画面】疯长的植物拱裂道路，发光菌群沿墙壁蔓延，鹿群穿过空荡街道，巨鸟停在废弃站牌上。

旁白：
全球生物都在变异。
它们开始拥有智慧，拥有判断，也拥有了自己的沉默。

广播：
“请全体市民减少非必要外出。地表生态活动范围正在扩大。重复，请减少非必要外出——”

旁白：
但奇怪的是，它们始终没有主动袭击人类。

旁白：
于是你们在恐惧里，勉强和这个陌生的新地球维持着一种脆弱的和平。

【场景 0-2】北郊高架桥下 / 雨后 / 黄昏
【画面】桥下积水反着灰白天光，一只受伤的狼伏在断裂护栏后，金色眼睛穿过阴影盯着你。
【音效】滴水声，远处金属板晃动声。

旁白：
你第一次见到那只狼的时候，地球还没有完全死去。

你（低声）：
“别动……我不是来抓你的。”

旁白：
它的后腿被割开一道伤口，毛发被血黏成一缕一缕。
它没有扑上来。
你也没有后退。

你：
“我给你包扎。你要是不同意，现在就咬我。”

【画面】你把医疗包放在地上，慢慢蹲下。

旁白：
它盯着你，像在衡量你值不值得信任。
很久之后，它没有动。

旁白：
那就是它的回答。

【画面】你替它清理伤口、缠上绷带。狼安静地看着你，呼吸沉重而温热。

你（心声）：
……原来狼的眼睛，不只是凶。

旁白：
后来你又见过它几次。
有时候是在林地边缘，有时候是在被藤蔓淹没的地铁口。
它不亲近你，也不远离你，只在你陷入危险时恰好出现。

旁白：
你曾经以为，只要这种默契还在，世界就还没有彻底坏掉。


【第一章：总攻之日】

【场景 1-1】城区撤离通道 / 黄昏转夜 / 混乱
【画面】人群奔逃，天幕裂开，舰群像钢铁海潮从云后倾泻而下。
【音效】爆炸、警报、哭喊、广播过载。

广播：
“请立刻前往最近的轨道撤离点！重复，请立刻——”

【画面】远处高楼被白光切开，整片城区起火。

旁白：
总攻开始的那一天，所有频道都在同一秒里失真。

你：
“别停下！往前跑！”

旁白：
城市防御系统只撑了不到二十分钟。
你回头时，地球像一颗正在燃烧的果实。

【场景 1-2】近地轨道避难所 / 紧急舱段 / 数分钟后
【画面】红灯狂闪，舱体剧烈倾斜，人群撞向舱壁。
【音效】金属扭曲、结构爆裂、孩子哭声。

旁白：
你以为高空至少是安全的。

旁白：
可避难所被击落，只用了十三分钟。

【画面】你被推进逃生舱，舱门闭合。

你：
“等等！后面还有人——”

【音效】推进器点火，舱体猛震。

旁白：
逃生过程中，外部攻击追上了你们。
第一下震动之后，对面抱着孩子的女人就死了。
第二下更近，整舱陷入黑暗。

【画面】红色指示灯映着漂浮的血珠。

旁白：
活到最后的，只剩你和那个设计师。

【场景 1-3】逃生舱内部 / 持续坠落
【画面】老人胸口被金属碎片切开，手却死死抓住你的手腕。

设计师：
“听着……舱底有一个生物方舟。”

你：
“什么？”

设计师：
“外星技术逆向出来的……只要送到预设节点启动，它就能把环境改回去。”

【画面】舱底紧急锁扣后，一具银灰色装置隐在阴影里，像活着的金属茧。

你：
“改回去？”

设计师：
“外星人就是靠环境改造杀了大多数人。”
“反过来用，它们活不下来。”
“旧的物种会回来……原来的地球，会回来。”

你：
“为什么是我？”

设计师：
“因为你还活着。”

旁白：
这是他留下的最后一句话。


【第二章：无声地表】

【场景 2-1】坠毁点 / 黎明前 / 死寂
【画面】逃生舱撕开一道口子，冷风灌入。尸体歪挂在支架之间。
【音效】远处低鸣，金属热胀冷缩的细响。

旁白：
逃生舱坠地时，你觉得自己像被整个世界摔进泥里。

【画面】你拖着伤腿，把生物方舟一点点从舱内拽出来。

你（喘息）：
“……再出来一点……再一点……”

旁白：
它比看上去更沉。
每挪动一寸，破裂的舱壁都会发出刺耳摩擦。

【画面】一块小金属片从舱边掉落。
【音效】叮。

【画面】废墟另一侧，几个细长的黑色身影同时抬头。

旁白：
你第一次见到自律机器人。

【画面】感应环收缩、亮起。
【音效】高频锁定声。

你：
“糟了——”

【画面】灼热射束擦着你肋侧掠过，地面炸开白烟。

旁白：
那一刻你记住了两件事。

旁白：
第一，这些机器人对震动和声音极其敏感。
第二，安静有时候比奔跑更能救命。

【场景 2-2】地表废墟 / 数日蒙太奇 / 昼夜交替
【画面】你匍匐前进、伏倒、等待；巡逻舰低空掠过；你借轰鸣冲过街口。
【音效】巡逻舰轰鸣，呼吸声，碎石轻滚。

旁白：
之后很长一段时间里，你只在巡逻舰经过时行动。

旁白：
白天，地面处在巡逻舰和机器人的交叉监控下。
夜里更糟，因为热源太明显。

你（心声）：
等它过去。
再等近一点。
现在，跑。

【画面】第三个黄昏，生物方舟表面亮起流动光纹，自行折叠变形成一辆低矮小车。
【音效】机械展开声，低频启动音。

方舟系统：
“移动形态已解锁。”

你：
“你总算舍得醒了。”

方舟系统：
“建议：低速、低震、低噪行进。”

旁白：
于是你学会了另一种节奏。
停。听。等。
等轰鸣覆盖世界，再把自己从阴影里推向下一个阴影。


【第三章：那间房子还在生活】

【场景 3-1】居民楼外 / 傍晚 / 静止
【画面】一栋孤零零的居民楼立在荒败城区中央，外墙剥落，楼体却完好。

旁白：
方舟给出的坐标，终点是一栋居民楼。

【画面】你把小车停进阴影，沿着楼道慢慢上行。
【音效】楼道风声，遥远的舰船低鸣。

【场景 3-2】某户人家 / 黄昏末尾 / 安静到不真实
【画面】门口两双拖鞋，桌上防尘罩，厨房挂着擦手巾，床头放着翻到一半的书。

旁白：
你推开门时，几乎以为自己走错了世界。

旁白：
这个房间保存得太完好了。
好像这里的主人只是临时出门，随时都会回来。

你（很轻）：
“……对不起。”

旁白：
你不知道这句话是在对谁说。
对这间房子。
对它曾经的主人。
还是对已经来不及挽回的人类生活。

【画面】你躺到床上，闭眼。
【音效】弹簧轻响，呼吸渐缓。

旁白：
撤离通道的尖叫、避难所里的红灯、设计师临死前的眼神，那只狼沉默的眼睛——
所有你不敢碰的回忆，一次性涌了回来。

【场景 3-3】客厅 / 夜色降临前 / 紧张
【画面】方舟在客厅中央展开，缩小成冰箱大小，正面弹出一排实体按钮。

方舟系统：
“请以节奏序列完成启动校准。”

你：
“……都世界末日了，你还要我玩音游？”

【画面】按钮依次亮起，节奏由慢到快。
【音效】按钮音、巡逻舰由远及近的轰鸣。

旁白：
第一组像心跳前奏。
第二组开始变快。
第三组快得几乎不讲道理。

你（咬牙）：
“来。”

【画面】你按下一个又一个按钮，额角全是冷汗。

旁白：
按到一半时，头顶那艘巡逻舰飞走了。

【音效】轰鸣骤然远去，只剩按钮与呼吸声。

旁白：
世界一下子安静得可怕。

你（心声）：
停下，前功尽弃。
继续，就会被听见。

【画面】你盯着最后几组光标，手指微颤。

你：
“……那就听见吧。”

【画面】最后一个按钮落下，方舟长鸣，一枚胶囊状发光物体从缝隙中升起，沉入顶部透明槽。

方舟系统：
“第一阶段启动成功。”

【音效】楼下玻璃连片震裂。

【画面】街道尽头，数个自律机器人抬起头，感应环亮成刺目的红。

你：
“来得真快。”


【第四章：狼群】

【场景 4-1】居民楼外街道 / 夜 / 追逐
【画面】方舟切回小车形态，你从楼道冲出，车体被射束连续命中。
【音效】射击、爆裂、警报、车体损坏提示。

方舟系统：
“外壳受损 47%。”
“外壳受损 63%。”
“外壳受损 81%。”

你：
“闭嘴，我看得见！”

旁白：
方舟没有攻击能力。
它只能带着你逃，被动挨打，直到报废。

【画面】街道前方被废墟堵死，左右巷道也有机器人逼近。

旁白：
你知道自己就要撑不住了。

【音效】远处，一声狼嚎。
接着是第二声，第三声，更多声。

【画面】灰色、黑色、银白色的狼影从楼影、塌陷停车场和断墙后扑出。

旁白：
一直以来，狼群都作为中立方而存在。
机器人从不主动攻击它们。

旁白：
可这一刻，它们突然扑向了你的追兵。

【画面】最前面的机器人停顿了一瞬，像命令链在犹豫：要不要攻击狼群。

旁白：
而狼群抢的，就是这一瞬间。

【画面】领头的狼跃上车侧，用身体狠狠撞了方舟一下，把你推向一条通往森林的裂口小路。

你：
“……是你？”

【音效】狼群撕扯金属、撞击传感器、扰乱阵列；高空巡逻舰掉头。

旁白：
你几乎是被狼群半推半护地带进了森林。

【场景 4-2】森林边缘 / 深夜 / 暂时安全
【画面】方舟小车冒着白烟停下，你跌进潮湿落叶里。狼群在周围安静地围成一圈。

旁白：
直到车体熄火，你才意识到自己还活着。

【画面】那只狼走到你面前，侧腹添了新伤，金色眼睛依旧很稳。

你：
“为什么……”

【停顿】

狼：
“因为你把它带到了这里。”

【画面】你猛地抬头。

你：
“……你会说话？”

狼：
“我们一直都会。”
“只是以前，你们没有时间听。”


【第五章：森林会说话】

【场景 5-1】林间空地 / 深夜 / 冷光微亮
【画面】雾气在树根间浮动，远处地平线下有极淡的光正在升起。

你：
“那是什么？”

狼：
“方舟已经开始改写环境。”
“但还没有结束。”

你：
“设计师说，它会把地球改回原来的样子。”

狼：
“那只是人类对它的理解。”

【画面】狼慢慢绕着你走了一圈，视线落在报废边缘的方舟上。

狼：
“你们把它当成武器，当成复仇，当成让一切倒退的方法。”
“可地球从来不在乎倒退。”
“它只在乎能不能活下来。”

你：
“你到底想说什么？”

狼：
“我们不是外星人的造物。”
“我们是地球被撕开以后，为了活下来，长出来的新东西。”

旁白：
夜风吹过树梢，整片森林都在低声作响。
像有无数双眼睛正在黑暗里看着你。

狼：
“如果你按人类最初的设定走完方舟，外星人会死。”
“我们也会死。”

你：
“……包括你？”

狼：
“包括我。”

【场景 5-2】更深处的森林 / 持续前行 / 神秘
【画面】狼群带你穿过树林，来到一处被根系与金属残骸缠死的巨大坑谷。坑中央，一座外星改造塔被古树根须贯穿。
【音效】低频共鸣，远处舰船回旋。

方舟系统：
“检测到全球环境回写进度 61%。”
“检测到最终模板未确认。”

你：
“最终模板？”

方舟系统：
“可选项如下：
一，原生地球模板回滚。
二，融合模板保留。”

【画面】你怔住。

你：
“还有第二种？”

狼：
“有。”
“不是回到过去。”
“是把过去留下来，交给未来。”

你：
“如果选第一种，人类也许能拿回原来的地球。”

狼：
“可那只是地球的尸体。”
“干净，熟悉，没有我们，也没有你失去的那些人。”

你：
“如果选第二种？”

狼：
“外星人的环境逻辑会被清除。”
“原有物种会回归，新的生命也会被允许留下。”
“但世界不会回到从前。”
“死去的人不会回来，倒塌的城不会自己站起来。”
“从今以后，人类必须学会和这个新的地球一起活。”

旁白：
你忽然想起那间保存完好的房间。
想起桌上的防尘罩、床头的旧书、厨房挂着的擦手巾。

旁白：
那些生活之所以珍贵，从来不是因为人类拥有过地球。
而是因为人类曾经认真地生活在地球上。

你：
“我想让灯重新亮起来。”
“不是把后来的一切都判死刑。”

【画面】狼静静看着你，像很久以前在高架桥下那样。

狼：
“那就选吧。”


【第六章：最后的校准】

【场景 6-1】外星改造塔遗骸 / 黎明前最黑时 / 决断
【画面】你把方舟拖到根系中央的凹槽前，透明槽中的发光胶囊缓缓升起。
【音效】系统启动声，四周树根共鸣。

方舟系统：
“最终校准开始。”
“请确认生态模板。”

你：
“确认：融合模板保留。”

方舟系统：
“确认收到。”
“请执行最终节奏同步。”

你：
“怎么又来？”

狼：
“因为心跳、脚步和风，本来就是最早的节奏。”

【音效】远处，巡逻舰轰鸣逼近；林外，自律机器人集群开始穿林推进。

旁白：
它们还是追来了。

【场景 6-2】同一地点 / 最终对抗 / 高潮
【画面】按钮再次亮起，比之前更快、更密，像暴雨坠下。
【音效】按钮提示音、巡逻舰低鸣、机器人锁定声。

方舟系统：
“同步开始。”

【画面】领头的狼抬起头，发出一声悠长的嚎叫。
狼群随之应和，节奏一层层叠上来。

旁白：
你忽然明白了。
此前让你暴露、让你恐惧、让你一次次濒死的声音——
此刻成了掩护你的东西。

你（咬牙）：
“好。再来。”

【画面】你跟着狼嚎与系统节拍不断按下按钮。
第一组。
第二组。
第三组。

【画面】机器人冲出林线，射束擦着树干掠过；狼群扑上去拖住它们；高空巡逻舰开始俯冲。

狼：
“别停！”

你：
“我知道！”

【音效】最后一串节拍像心脏濒死前的狂跳。

旁白：
你的手在抖。
肩伤裂开。
视野边缘开始发白。

你（心声）：
再一下。
最后一下。

【画面】你按下最终按钮。
与此同时，领头的狼发出最后一声长嚎。

【音效】整座森林与大地同时回应，像有什么东西终于在地心深处醒来。

方舟系统：
“校准完成。”
“地球环境回写开始。”

【画面】发光胶囊碎裂成无数细小光点，沿树根、土壤、建筑残骸和空气本身扩散。

【画面】天空中，巡逻舰的外壳先是失去光泽，随后像被抽空一样失速坠落。
地面上，机器人的感应环依次熄灭，动作僵住，最后成排倒下。

【画面】外星改造塔从中心裂开，像死去的昆虫蜕壳。
病白色天空一层层褪去，露出更深、更干净的蓝。

旁白：
那不是回到过去。
那是地球把寄生在自己身上的东西，一点一点剥了下来。


【第七章：没有回去的世界】

【场景 7-1】森林 / 黎明 / 风重新开始流动
【画面】阳光从林叶间漏下来，落在你的手背上。那是一种久违的、真正属于地球的暖意。

你（发怔）：
“结束了吗？”

狼：
“外星人结束了。”
“世界没有。”

【画面】林间雾气散开，远处传来鸟鸣。不是变异后的尖啸，而是你记忆里更普通、更轻的鸣叫。

旁白：
旧的物种回来了。
新的生命没有消失。
森林、城市和废墟一起呼吸，像刚从长期窒息里醒来。

你：
“人类……还剩下多少？”

狼：
“比你以为的多。”
“比从前的世界，当然少得多。”

【音效】你腰间那部几乎报废的短波机，忽然发出断断续续的噪声。

短波机：
“……这里是东部地下避难站……”
“……如果还有幸存者，请回应……”

【画面】你愣了几秒，手指几乎握不住机器。

你（发抖）：
“这里是地表幸存者。”
“这里是地球。”
“我能听见你。”

【场景 7-2】同一地点 / 片刻之后 / 安静
【画面】你放下短波机，转头看向那只狼。

你：
“谢谢你。”

狼：
“你也救过我一次。”

你：
“我只是给你包扎了伤口。”

狼：
“对快死的东西来说，包扎和拯救没有区别。”

旁白：
你忽然想起高架桥下那个雨后的黄昏。
原来从那时候开始，一切就已经悄悄连在一起了。


【终章：巡逻舰不再经过】

【场景 终-1】数月后 / 北郊新聚居地 / 清晨
【画面】巡逻舰残骸半埋在草坡里，旧居民楼的窗边挂起了新晾的布。人类幸存者在废墟上搭建新的住处，远处鹿群从林边经过，彼此保持距离，但不再惊逃。
【音效】风声、木材敲击声、远鸟鸣。

旁白：
后来你们找到了更多活下来的人。
海上的漂流舱，地下避难站，被山体遮蔽的小型据点。

旁白：
没有谁真正拿回了从前的世界。
可至少，世界不再只剩下死亡。

旁白：
人类开始学习新的事情。
如何在森林边界点灯。
如何给会思考的兽群留下水源。
如何在旧城市的废墟上，和新的地球重新谈判。

【场景 终-2】高架桥旧址 / 傍晚 / 熟悉的风
【画面】你独自站在桥下，和很久以前一样。积水已经清了，桥墩缝里长出柔软的草。

【音效】脚步踩过碎石。

【画面】那只狼从林影里走出来，比从前更安静，也更像一场没有结束的梦。

你：
“我有时候还会想，如果那天我在桥下转身走了，会怎么样。”

狼：
“那你就不会活到现在。”

你：
“原来你从那时起，就知道会有今天？”

狼：
“不。”

【停顿】

狼：
“从那天起，我只知道——人类里还有人会停下来。”

【画面】风吹过高架桥，狼转身走回林中。

你：
“你还会再来吗？”

【画面】狼没有回头，只是尾巴轻轻摆了一下，就像很多很多天以前一样。

旁白：
你知道那不是告别。

旁白：
从今以后，这颗星球不会再只属于人类，也不会再被外来者支配。
它会属于一切愿意活下去、并愿意让别的生命一起活下去的东西。

旁白：
风穿过城市的时候，你还是会下意识抬头。

旁白：
只是如今，经过你头顶的，不再是巡逻舰。

【画面】成群候鸟掠过恢复湛蓝的天空。

旁白：
而是春天。
